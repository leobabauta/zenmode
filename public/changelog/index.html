Changelog & Roadmap

What's new in zenmode and what's coming next.

Changelog

v0.9 — Current Release

  • Reminders — Type @3pm, @tomorrow 2pm, or @mon 9am to schedule a reminder. Hidden until the time arrives, then pops into Today with a notification
  • Recurring Reminders — Type every day @3pm or every monday @4pm to create recurring reminders
  • Smart Time Detection — Type a time at the end of a task (e.g. "Call dentist 3pm") and you'll be asked if you want to set a reminder. Supports shorthand like 3p, 4:30a
  • Reminder Bell Icon — Reminders show a bell instead of a checkbox (green checkmark when done), with the time shown inline
  • Recently-Fired Highlight — Reminders glow amber for 10 minutes after firing, then return to normal
  • Pending Reminders View — Bell icon on each day in Timeline and Today shows upcoming reminders with delete button
  • Desktop App (Tauri) — Lightweight native app for macOS and Windows (13 MB vs 236 MB Electron). Global quick-add hotkey, system tray, persistent login
  • Ritual Snooze Sync — Snoozing or skipping a ritual prompt now syncs across all browsers and devices

v0.8

  • Mobile App (Android) — Native React Native app with all core features: Today, Timeline, Inbox, Lists, Focus Timer, Settings
  • Pull-to-Refresh — Pull down on any mobile screen to force sync
  • Context-Aware Add Task — The add button shows where a task will be added (Today, Inbox, or current list) with a dropdown to switch
  • Timeline Past Days — Scroll back 14 days in the mobile timeline to see previous tasks
  • Morning Nudge Notification — Configurable daily reminder to review your plan, with time picker in Settings
  • Focus Timer Notification — Push notification when your focus timer completes, even if the app is backgrounded
  • Mobile Reminder Notifications — Push notifications for reminders on mobile, scheduled via expo-notifications

v0.7

  • Accent Color Themes — Choose from 10 color presets in Settings
  • Email + Password Auth — Sign up and sign in with email and password, alongside Google OAuth
  • Priority Star Sparks — Completing a priority task triggers an explosion of star-shaped sparks
  • Copy Tasks as Markdown — Select tasks and press Cmd+C to copy as formatted Markdown
  • Today View Improvements — Violet glow card, auto-select first incomplete task, greeting inside card
  • Task Focus Mode — Full-screen task view with Markdown-rendered notes
  • Smart Sidebar — Today auto-collapses sidebar, other views follow your preference

v0.6

  • Email-to-Task — Forward emails to create tasks in your Inbox automatically
  • Subtasks — Option+drag to nest tasks, with collapsible display
  • Drag to Sidebar — Drag tasks to Inbox, Later, Archive, custom lists, or labels
  • Command Palette — Quick access to navigation and task creation with Cmd+K
  • Sidebar Reordering — Drag to reorder nav items and labels
  • Completion Confetti — Full-screen celebration when all tasks are done

v0.5

  • Daily Planning & Review Rituals — Guided morning workflow and evening reflection
  • Weekly Planning & Review — Configurable weekly rituals with day and time settings
  • Focus Timer — Built-in Pomodoro timer with analog dial and session tracking
  • Timeline View — Week at a glance with drag-and-drop between days
  • Inbox, Later, Custom Lists, Labels — Full task organization system
  • Cross-Device Sync — Real-time sync across all devices via Supabase
  • Dark Mode, Keyboard Shortcuts, Recurring Tasks, Stats
